
# Readme.md



## Descripcion de EcoRecicla

EcoRecicla: Es una aplicación móvil diseñada para fomentar y facilitar el reciclaje responsable. Su objetivo es promover la conciencia ambiental y ayudar a reducir la cantidad de residuos que terminan en los vertederos.
